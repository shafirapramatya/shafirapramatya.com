The Challenge

Online storefronts make a lot of data rows, but finding exactly where a store is losing cash requires sorting metrics carefully across regions, pricing, and item sets.

The Approach

Built sorting scripts in Google Colab to tidy up over 3,500 rows of store orders, then added formulas to map total profit splits and generate comparison charts for regional trends.

What This Project Looks Into

  • Which store items make the best returns vs which items stay on shelves too long?
  • How much do delivery fees reduce our overall shop returns?
  • Where do holiday demand spikes create low stock risks?
  • Which buyer regions show the best performance scores for store spending?

The Challenge

Country growth plans are linked with public lifestyle indicators, but because these data fields are kept on separate websites, it is hard to map how they affect each other.

The Approach

Merged international stats from the World Happiness files and country integrity score logs in Python, building trend graphs and cross-region overview models.

What This Project Looks Into

  • Does a country's transparency index show a clear connection with public lifestyle happiness?
  • Which institutional metrics move closely alongside public well-being markers?
  • How did the connections between corruption and happiness change from 2015 to 2020?
  • Which continents show the highest or lowest connection lines for these markers?
