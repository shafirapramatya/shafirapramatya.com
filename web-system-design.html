Project Checklist:

  • The Problem: Finding out why local cultural workshops and small craft shops struggle to find customers online.
  • Understanding Users: Grouping what international tourists want versus what local communities need from a cultural workshop.
  • Pitch Prep: Putting together interactive feedback frames that we used for our international pitch contest reviews.

Validated User Segments

A quantitative and qualitative survey via Google Forms grouped target users into four segments, each with distinct needs the platform's information architecture had to serve.

Locals

Seeking accessible cultural learning

Educators

Need structured cultural curriculum to teach from

Expats / Travelers

Want authenticated, localized cultural tour guides

Cultural SMEs

Lack exposure to market goods beyond local circles

Designing how information links up across multiple vendor setups. I mapped out the connections between shop items, class entries, and safe login sessions while keeping the code efficient and error-free.

Database Features:

Built to track user accounts, manage payment balances between partner stores, and store secure browsing history smoothly without cluttering up the server logs.
